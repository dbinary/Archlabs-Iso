# Lila-Gtk
Gtk2/3 for XFCE4 

<P>Download Zip or clone repo</P> 
<P>extract in case of zip file </P>
<P>move to ~themes               for your user</P>

<P>select Lila-Gtk theme </P>
<P>Enjoy.</P>
